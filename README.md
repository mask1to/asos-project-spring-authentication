# Spring Boot & Spring Security - 2FA

The repository contains implementation of **2FA via Spring Boot & Spring Security**.

**Main target of the project** is to make the web environment more secure, that means, if you will login to website, which has 2FA, after entering your password, you will be asked to approve push notification or provide one-time password code, which is being generated in the mobile app - Authy - also used in this project.
